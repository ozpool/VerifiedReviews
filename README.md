# VerifiedReviews

Receipt-gated on-chain reviews for local businesses. A non-transferable
VisitProof SBT (ERC-5192) minted at point-of-sale gates reviews behind a
60-day recency check, enforced **on-chain** — not by the server.

**Stack:** pnpm monorepo · Solidity + Hardhat · Express + TypeScript · Next.js + Wagmi · MongoDB Atlas · Arbitrum Sepolia

See `archi.md` for the architecture and `git.md` for the delivery plan.
